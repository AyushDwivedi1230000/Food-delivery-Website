HOW TO RUN
================================================================================

1. Install Dependencies:
   npm install

2. Configure Environment Variables:
   - backend/.env (MONGO_URI, JWT_SECRET, etc.)
   - frontend/.env (VITE_API_URL)

3. Start MongoDB:
   mongod

4. Seed Database (Optional):
   npm run seed

5. Start Development:
   npm run dev

   This starts:
   - Frontend: http://localhost:5173
   - Backend: http://localhost:5000
